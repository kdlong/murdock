LaTeX Software

Free or nearly free software for LaTeX:

  • MikTeX
    A free version of TeX for the PC. The full download has everything and is very large.
  • WinEdt
    A good editor which is great to use with TeX files; it comes with a nice interface to the MikTeX system. You can try the program for a while but to use it longer it costs $40. If you can afford it, this is a good program to have.

Not-So-Free LaTeX

It is true that the basic Tex and LaTeX programs were made free to the world, but a few companies have issued the basic software along with their own programs for editing the files and viewing and printing the results, and for installing all the software. Don't look for academic pricing on these because most of the people who use LaTeX are known to be academic.
  • PC-TeX
  • Y and Y TeX
  • Visual TeX
